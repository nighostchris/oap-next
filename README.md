# oap-next (Last Update: 15/10/2019 16:05)
oap-next = Next.js implementation of Online Assessment Platform.

### Installation and Run project
```sh
> git clone https://gitlab.com/hkust/cse/oap-next.git
> cd oap-next
> npm install
> npm run dev
```
